rglab::software

Raphael Gottardo's Research Lab

R packages

Our group currently maintains the R packages listed below, which can be downloaded directly from Bioconductor.

  • Packages for flow cytometry: flowCore, flowWorkspace, flowStats, flowQ, flowViz, flowClust, flowMerge, flowTrans, QUALIFIER
  • Packages for ChIP-Seq and motif analysis: PICS, PING, rGADEM, MotIV
  • Packages for chip-chip and tiling arrays: BAC, rMAT
  • Packages for differential gene expression: rama, bridge

Labkey Server

We also implement some of our statistical methods and tools in the LabKey Server.

GitHub

Most of our work is openly accessible on our github page.
